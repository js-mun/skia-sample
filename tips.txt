## gdb core
Linux 배포판과 설정에 따라 core 파일 위치가 다를 수 있음.
```
cat /proc/sys/kernel/core_pattern
```
임시로 core 파일 직접 생성
```
sudo systemctl stop apport.service   # Ubuntu에서 apport 임시 비활성화
ulimit -c unlimited                  # core dump 허용
./sample                             # crash 발생시키면 core 파일 현재 디렉터리에 생성
```
